# Ice-cream-Parlour-Website-Using-Node.js
Ice Cream Paradise Website
This project is a fully responsive website for an ice cream shop called Ice Cream Paradise, featuring a modern UI with flavor catalogs, ordering system, and backend server.
​

Features
Interactive menu with flavor categories: Classic (Vanilla, Chocolate, Strawberry, Butterscotch), Fruit-Based (Mango, Coconut, Guava, Lychee), and Chocolate-Based (Dark Chocolate, Belgium Chocolate, Brownie Fudge, Nutella).
​

Order page with quantity selectors, delivery options, order summary, payment methods, and special combo offers.
​

Home, Explore, Contact pages with contact form, FAQ, and filter functionality for flavors.
​

Premium quality theme emphasizing organic products, wide variety, and home delivery.
​

Node.js HTTP server handling routes for all pages using core modules only.
​

Tech Stack
Frontend: HTML5, CSS3 (style.css with responsive design and animations), vanilla JavaScript for filters and interactions.
​

Backend: Node.js server (index.js) serving static files and dynamic pages.
​

Assets: High-quality images for 15+ ice cream flavors, logo, hero banner.
​

Dependencies: None (core Node.js modules), Font Awesome for icons.
​
Quick Start
Ensure Node.js >=12.0.0 is installed.
​

Run npm install (no external deps needed).

Start the server: npm start or node index.js.
​

Open http://localhost:8000 (or configured port) in browser.
​

Access pages: Home, Menu (index.html), Explore, Order Now, Contact.
