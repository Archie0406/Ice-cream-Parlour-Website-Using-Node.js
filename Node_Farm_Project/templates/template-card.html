Product Card Component Template

This is a reusable product card template component documentation.

HTML Structure:

<div class="product-card">
    <img src="[IMAGE_PATH]" alt="[PRODUCT_NAME]" class="product-card-image">
    <div class="product-card-content">
        <p class="product-card-category">[CATEGORY]</p>
        <h3 class="product-card-name">[PRODUCT_NAME]</h3>
        <p class="product-card-description">[DESCRIPTION]</p>
        <div class="product-card-footer">
            <span class="product-card-price">[PRICE]</span>
            <button class="product-card-btn">View Details</button>
        </div>
    </div>
</div>

CSS Classes:

  • .product-card - Main container
  • .product-card-image - Product image
  • .product-card-content - Content wrapper
  • .product-card-category - Category label
  • .product-card-name - Product name
  • .product-card-description - Product description
  • .product-card-footer - Footer with price and button
  • .product-card-price - Price display
  • .product-card-btn - Action button

Variables to Replace:

  • [IMAGE_PATH] - Path to product image
  • [PRODUCT_NAME] - Name of the product
  • [CATEGORY] - Product category
  • [DESCRIPTION] - Product description
  • [PRICE] - Product price
